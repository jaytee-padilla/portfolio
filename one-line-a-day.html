One Line A Day

Frontend Engineer & Project Lead

Tech Stack: React, React Router, Axios, Semantic UI, Styled Components, SCSS

A platform for users to record and retrieve their thoughts.

The One Line A Day application was built from the ground up by a team of 5 software engineers over the course of 4 days.

    I was responsible for:
  • - Leading a team of 3 Frontend Developers, 1 Backend Developer, and bridging the gap between the two disciplines
  • - Handling all github pull requests and merge conflicts between the frontend developers
  • - Designing the UI/UX
  • - Hooking up all frontend components to the backend
  • - Implementing dynamic routing, protected routing, user registration, and logging in/out
  • - Refactoring of components' code, styling code, and project file structure
